# Vue.extend用法 注册组件

`Vue.extend` 是构造一个组件的语法器。

## 用法

Vue.extend ( options )，options 是对象；
使用基础Vue构造器，创建一个子类，参数是一个包含组件选项的对象，data选项是特例，它必须是函数。
