# typora画流程图、时序图(顺序图)、甘特图 

markdown做为笔记记录，在方便不过，这里记录经常要用的流程图画法模版 , 感觉也不是很方便,还是图片拖拽笔记方便

## 1.横向流程图源码格式：

\```mermaid

graph LR

A[方形] -->B(圆角)

​    B --> C{条件a}

​    C -->|a=1| D[结果1]

​    C -->|a=2| E[结果2]

​    F[横向流程图]

\```

## 3.标准流程图源码格式：

```
​```flow
st=>start: 开始框
op=>operation: 处理框
cond=>condition: 判断框(是或否?)
sub1=>subroutine: 子流程
io=>inputoutput: 输入输出框
e=>end: 结束框


st->op->cond
cond(yes)->io->e
cond(no)->sub1(right)->op
​```
```

## 5.UML时序图源码样例：

\```sequence

对象A->对象B: 对象B你好吗?（请求）

Note right of 对象B: 对象B的描述

Note left of 对象A: 对象A的描述(提示)

对象B-->对象A: 我很好(响应)

对象A->对象B: 你真的好吗？

\```
